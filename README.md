# Indian Stock Signal Generator

A Streamlit-based stock analysis dashboard that uses Moving Averages and RSI to generate Buy/Sell signals.

## Features
- Stock price visualization
- Moving Averages (MA10, MA50)
- RSI Indicator
- Buy/Sell/Hold signals
- Clean dashboard UI

## How to Run
```bash
pip install -r requirements.txt
streamlit run app.py
